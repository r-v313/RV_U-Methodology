# RV_U Methodology — Complete Bug Bounty Hunting Platform

> A professional, single-page web application with 64 sections covering every aspect of bug bounty hunting methodology.

---

## ✨ Features

- 🎯 **Domain Input System** — Add targets by typing, pasting, or uploading a `.txt` file
- 🔍 **Auto-detection** — Automatically identifies wildcards, domains, IPs, CIDR ranges, and URLs
- 🏷️ **Visual tagging** — Color-coded chips show each target's type
- 🚀 **Start Methodology** — Scrolls to the right section based on your target type
- 🔎 **Global Search** — Instant search across all 64 sections (`Ctrl+K`)
- 📋 **Copy buttons** — Every code block has a one-click copy button
- ✅ **Progress tracking** — Mark sections as done with a progress bar
- 📱 **Responsive design** — Works on desktop, tablet, and mobile
- 🌑 **Dark hacker theme** — Neon green / cyan / purple accents

## 🚀 Usage

1. Open `index.html` in any modern browser — no installation required
2. Add your targets in the **Target Input System** at the top
3. Click **Start Methodology** to jump to the right starting section
4. Use `Ctrl+K` to search across all sections
5. Mark sections as **Done** to track your progress
